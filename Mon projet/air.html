L’air et l’eau – Niveau DEF

Leçon simple et claire pour débutants — propriétés, composition, cycle

1. L’air : qu’est-ce que c’est ?

L’air est un mélange de gaz qui nous entoure. Il est invisible mais indispensable à la vie.

  • Dioxygène (O₂) : environ 21 % → permet la respiration.
  • Dioxyde de carbone (CO₂) : moins de 1 % → rejeté par les êtres vivants.
  • Diazote (N₂) : environ 78 % → gaz principal de l’air.

2. Propriétés de l’air

  • L’air a une masse.
  • L’air exerce une pression.
  • L’air peut être comprimé.
  • L’air occupe un volume.

3. L’eau : importance

L’eau est indispensable à la vie : humains, animaux, plantes en ont besoin.

On la trouve dans les mers, rivières, lacs, glaciers et même dans l’air sous forme de vapeur.

4. Les états de l’eau

  • Solide : glace
  • Liquide : eau
  • Gazeux : vapeur d’eau

5. Le cycle de l’eau

  • Évaporation : l’eau se transforme en vapeur.
  • Condensation : la vapeur devient nuage.
  • Précipitations : pluie, neige, grêle.
  • Ruissellement : l’eau retourne vers les rivières et océans.
